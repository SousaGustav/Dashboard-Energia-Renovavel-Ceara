# Dashboard de Energia Renovável no Ceará
## Sobre o Projeto

Este projeto consiste em um dashboard interativo desenvolvido para apresentar dados relacionados às energias renováveis no estado do Ceará. O objetivo é transformar informações em visualizações claras e intuitivas, facilitando a análise do crescimento energético sustentável na região.

O dashboard reúne indicadores importantes sobre fontes renováveis, permitindo uma navegação dinâmica e uma experiência visual moderna.

## Temas Abordados
- Energia Solar 
- Energia Eólica 
- Produção Energética
- Sustentabilidade
- Crescimento do Setor Energético
- Indicadores e Estatísticas
- Visualização de Dados

## Objetivo

O principal objetivo deste projeto é demonstrar, de forma visual e acessível, a importância das energias renováveis no Ceará, destacando o crescimento sustentável e o potencial energético do estado.

Além disso, o projeto também serve como prática de desenvolvimento front-end e visualização de dados.
